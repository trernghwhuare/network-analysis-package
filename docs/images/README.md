# Sample Images

This directory contains sample images for the README.md file. 

To add actual sample results to the documentation:

1. Run the analysis on your network data
2. Copy the generated plots to this directory
3. Rename them to match the references in README.md:
   - `sample_regression.png`
   - `sample_distributions.png`
   - `sample_violins.png`
   - `sample_clustermap.png`
   - `sample_combined_heatmaps.png`
   - `sample_ei_stacked.png`

The images should showcase typical outputs from the analysis tools to help users understand what to expect.
